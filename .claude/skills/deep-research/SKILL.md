## 手順

以下のコマンドを使ってDeep Researchを実行できます。`<query>`の部分にリサーチしたい内容を入力してください。クエリを作成するのに情報が不足している場合は、ユーザーに聞き返してください。

```
$ mise run run_research <query> &
```

コマンドを実行すると、数十分後に結果が自動で保存されます。

コマンドの実行は時間がかかるので、最後に`&`を付けてバックグラウンドで実行してください。このスクリプトは実行完了すると自動でログと通知を行います。実行完了を待つ必要はありません。次のステップに進んでください。
